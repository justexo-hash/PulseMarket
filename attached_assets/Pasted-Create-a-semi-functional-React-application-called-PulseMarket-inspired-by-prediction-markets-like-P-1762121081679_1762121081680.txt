Create a semi-functional React application called Polymeme, inspired by prediction markets like Polymarket. The app will be a single-page application (SPA) that uses react-router-dom for navigation. It will display a list of markets, allow users to click for details, and simulate creating a new market.

2. Main App Component (src/App.jsx)
This component will be the "brain" of your app.

Import mockMarkets and store them in state: const [markets, setMarkets] = useState(mockMarkets);

Set up react-router-dom (BrowserRouter, Routes, Route).

Include the Header component on all pages.

Define a function handleCreateMarket that takes a newMarket object and adds it to the markets state (e.g., setMarkets([...markets, newMarket]);).

3. Routing (src/App.jsx)
Set up the following routes:

/: Renders the MarketList component.

/markets/:id: Renders the MarketDetail component.

/create: Renders the CreateMarket component.

4. Components (src/components/)
Create a components folder in src.

Header.jsx

A simple navigation bar.

Shows the name "Polymeme".

Uses the Link component from react-router-dom to link to:

"Markets" (/)

"Create Market" (/create)

MarketList.jsx (Homepage)

Receives the markets array as a prop.

Maps over the array and renders a MarketCard component for each market.

MarketCard.jsx

A presentational component that receives a single market object as a prop.

Displays the question, category, and probability (e.g., "65%").

The entire card should be wrapped in a Link component that navigates to /markets/${market.id}.

MarketDetail.jsx

Uses the useParams hook from react-router-dom to get the id from the URL.

Receives the full markets array as a prop.

Finds the correct market: const market = markets.find(m => m.id === parseInt(id));

Displays the full market question.

Semi-functional: Shows two buttons: "Bet Yes" and "Bet No".

On click, these buttons just trigger a JavaScript alert('Your mock trade has been placed!');.

CreateMarket.jsx

Receives the handleCreateMarket function as a prop.

Uses useState to control a simple form with inputs for "Question" and "Category".

Uses the useNavigate hook from react-router-dom.

On form submit:

Prevents the default form action.

Creates a new market object (don't forget a new id!).

Calls the handleCreateMarket prop with the new object.

Uses Maps('/') to redirect the user back to the homepage.
